1Ô∏è)Problem Statement
Elderly individuals often face challenges in managing their daily medications and monitoring their health conditions. Memory issues, complex prescriptions, and lack of supervision can result in missed doses, incorrect medication intake, and delayed medical intervention.

Many seniors live independently or remain alone for long hours, increasing the risk of unattended emergencies. There is a need for a reliable digital system that helps elderly individuals manage medications efficiently, monitor vital health parameters, and ensure timely communication with caregivers.

2Ô∏è)Proposed Solution

SENIOR SAFE is a digital healthcare support system designed specifically for elderly individuals.
The system provides:
    Automated medicine reminder system
    Health monitoring dashboard (BP, Sugar, Heart Rate tracking)
    One-click Emergency SOS alert
    Caregiver monitoring portal
    Elder-friendly and simple interface

The frontend is developed using Streamlit for easy and clean UI
The backend is built using FastAPI, providing high-performance
